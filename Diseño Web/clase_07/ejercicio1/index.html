Ya hemos estudiado la aplicación de estilos CSS a listas en diversos artículos de DesarrolloWeb.com, en el taller de CSS y en los distintos manuales que tenemos sobre la materia. En concreto, hay un artículo que vamos a completar con este otro taller de CSS, en el que vimos cómo Crear una barra de navegación vertical con listas y CSS. En este caso veremos como crear una barra de navegación, pero en horizontal. Además veremos como alterar los estilos cuando se pasa el ratón por encima de los enlaces.

Básicamente, para crear una barra de navegación vertical por listas, tenemos que conseguir que los elementos de la lista se pongan uno detrás de otro en la misma línea (el comportamiento normal de las listas es que aparezcan los elementos uno debajo del otro en distintas líneas). Además, cada elemento será un enlace y al pasar el ratón sobre ellos, debe resaltarse de alguna manera el elemento, lo que aportará el dinamismo a la barra de navegación. Todo esto se puede hacer con CSS. Veremos cómo a lo largo de este artículo.
